## ✨ Features

- 📖 View all books from database
- ➕ Add new books
- ✏️ Update existing books
- ❌ Delete books
- 👤 User Portal
- 🛠️ Employee Portal
- 🔐 Admin Portal
- 📊 Popular Books Histogram
- 🕒 Track Last 5 Added Books
- 🎨 Responsive modern UI
- ⚡ Fast frontend using Vite + React
- 🗂️ Organized component structure

## 🚀 Installation

```bash
# Clone Repository
git clone https://github.com/your-username/serephina-library.git

# Open Folder
cd serephina-library

# Install Dependencies
npm install

# Run Development Server
npm run dev
```
